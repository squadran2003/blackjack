# Blackjack

A famous card game built in Python.

## Features

- Play against the computer
- Simple command-line interface
- Follows standard Blackjack rules

## Requirements

- Python 3.x

## Installation

Clone the repository:

```bash
git clone /home/andy/Desktop/blackjack
cd blackjack
```

## Usage

Run the game:

```bash
python blackjack.py
```
